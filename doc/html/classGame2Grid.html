ExplorePath: Explores a path (N,S,E,W,NE,NW,SE,SW) to tell if any discs can be flipped if move (x0,y0) is played.

Parameters
x0
y0
x_direction{-1,0,1} to indicate in what increment we should move along the rows
y_direction{-1,0,1} to indicate in what increments we should move aling the columns

starting from x0,y0 proceed in increments of (+x_direction,+y_direction) and check:
If slot is empty–>empty return vector and return
If slot has a disc of same color –> return
If slot has a disc of opposite color –> add (x_slot,y_slot) to return vector
If end of board is reached without reaching slot of same color first –>emtpy return vector and return

Returns
vector of all discs that can be flipped in specified path. if vector is empty, nothing can be said about legality of the move, since other paths have to be checked

◆ PlayTurn

void Game2Grid::PlayTurn ( )
slot

PlayTurn: Slot connected to click signal for all ReversiSlots in grid.

Checks if x and y coords of clicked slot are in all_valid_moves.
If not, return
If it exists, call ValidMove(x,y) to decide on what discs must be flipped
For each disc in returned vector, change color to its opposite color (black to white and vice versa)
Update static variable turn
Update all_valid_moves by calling AllValidMoves
If updated all_valid_moves is empty, the next player cannot make a move
Reset turn to the player who just played, and call AllValidMoves
If updated all_valid_moves is again empty, the current player cannot make a move as well, meaning both players can't play
Game has ended
If so, compute the number of placed white discs and the black ones
The player having more discs is the winner

vector< vector< int > > Game2Grid::ValidMove ( int  x,
int  y 
)

ValidMove: Function checks if a ReversiSlot at position (x,y) can be played as a turn in the game.

For a move to be valid, it must lead to at least 1 of the opponents discs to be flipped.
This function checks in all 8 possible path (N,S,E,W,NE,NW,SE,SW) if such a disc exists

Returns
vector of all discs that should be flipped if the move at (x,y) is played. If vector is empty, move is illegal
